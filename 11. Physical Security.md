- General Security can be categorized in 3 ways
   - Physical
       - Security Guards, cameras, alarms
   - Technical
       - Firewall, IDS, AV
   - Operational
       - Controls, policies
       
- Physical Security Prevents
    - Unauthorized access to a computer system
    - Stealing of data from systems
    - Corruption of data
    - Loss of data from natural causes
    
- Physical Security Threats
   - Theft
   - Vandalism
   - Natural Causes
   
- Physical Site Counter Measures
    - Lock rooms
    - Setup surveillance
    - Secure work stations
    - Protect portable devices
    - Backups
    - Disable removable media
    - Protect printers
    - Identity Security
    - Tail gaters
